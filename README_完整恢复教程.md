                        二、新机器安装步骤
================================================================================

【前置要求】
- Windows 10/11 系统
- Python 3.12+
- Ollama (用于向量)

【步骤1：安装Python依赖】
pip install fastapi uvicorn requests pydantic

【步骤2：安装Ollama】
1. 下载 Ollama: https://ollama.com
2. 安装后运行: ollama serve --port 11435
3. 拉取向量模型: ollama pull bge-large

【步骤3：安装uvx (MiniMax MCP)】
1. 安装uv: pip install uv
2. uvx路径: C:\Users\Administrator\.local\bin\uvx.exe

【步骤4：恢复数据】
1. 将备份的 data 目录复制到:
   C:\Users\Administrator\.openclaw\workspace\ai-web-chat\data\

【步骤5：启动猪猪】
cd C:\Users\Administrator\.openclaw\workspace\ai-web-chat\backend
python main.py

前端: http://localhost:5180
后端: http://localhost:5181
